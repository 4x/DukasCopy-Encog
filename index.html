DukasCopy-Encog by fxmozart

Trading at Dukascopy with Encog neural framework

Retrieves data for encog neural network analysis

Dependencies

JForex ( Maven : http://www.dukascopy.com/client/jforexlib/publicrepo/ ) Encog Java Core framework : ( https://github.com/encog/encog-java-core )

Install

Open the project , set it all up by adding JForex + Encog (see above links) And run...
